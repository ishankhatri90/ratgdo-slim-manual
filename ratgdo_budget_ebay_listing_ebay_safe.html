Correct Wiring Reference

This version uses the included 3-conductor cable with red, green, and black conductors. Wire the opener side as shown below.

Wire Color Garage Opener Terminal Purpose Controller Side Note
Red RED / CTRL terminal Open / close control line Connects to the RATGDO red / middle connection point per the included diagram.
Green WHITE / GROUND terminal Common / ground Connects to the RATGDO green connection point per the included diagram.
Black BLACK / OBSTRUCTION terminal Safety sensor / obstruction line Connects to the RATGDO black connection point per the included diagram.
Wire before powering the controller. Do not plug in USB power until all three opener-side connections are verified. Incorrect wiring may damage the controller or opener.

Basic Installation Steps

  1. Turn off power to the garage door opener.
  2. Connect the included 3-wire cable to the opener terminals: red → RED/CTRL, green → WHITE/GROUND, black → BLACK/OBSTRUCTION.
  3. Verify the RATGDO-side wire order using the included photo/diagram.
  4. After wiring is confirmed, connect USB-C power.
  5. Configure Wi-Fi and firmware using the selected setup method.
  6. Test door, light, and obstruction status before relying on automations.
